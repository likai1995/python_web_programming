### 软件开发的架构<br>
1.C/S架构：即客户端/服务器端架构,需要用户安装exe程序才能运行<br>
2.B/S架构：即浏览器端/服务器端架构,不需要用户额外安装exe程序，在浏览器上经过http请求资源<br>
